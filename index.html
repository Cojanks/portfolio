Client Brief

The client, a public school educator, needed an intuitive and robust system to easily organize students into class periods, see student grades, and view, edit, and add individual student assignments. Student parents have ability to sign in to this application and see their particular student.

System
Project built in Node using the Express Node framework.
Intuitive
Responsively designed using Semantic-UI framework and rendered using EJS to inject database data.
Organize
Database built using MongoDB and the Mongoose Node package. Schemas include object references which are populated during the GET response.
View / Edit / Add
Using RESTful Routing and CRUD techniques, the teacher has functionality to add and subtract from the database at will.
Sign In
Using the Passport Node package as middleware, users (parents) are authenticated and shown only the student information which is associated with that user.
